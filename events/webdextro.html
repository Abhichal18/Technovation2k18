Competition will be held in two rounds.

ROUND-1: Qualifier Round(45 min)
1. 30 MCQs to be solved in 45 min based on basic concept of HTML, CSS and JAVASCRIPT.
2. Solo Candidates can take part in competition.
3. If someone is using any unfair means , he/she will be immediately diaqualified from the competition
4. Top ten students will be selected for the next round.

ROUND-2: Final Round(1 hr)
1. A picture will be provided and you have to make page that resemble it.
2. Participant can use HTML, CSS and JAVASCRIPT only.
3. No framework and jQuery library is allowed.
4. Coordinators decision will be final.
